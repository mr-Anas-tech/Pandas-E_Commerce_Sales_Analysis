# Pandas-E_Commerce_Sales_Analysis
  ****The project uses three datasets:****
- `customers.xlsx`: Contains customer details (ID, age, gender, city, signup date)
- `products.xlsx`: Product catalog with category and name
- `orders.xlsx`: Order details including quantity, price, and dates
- ** **- Data inspection and cleaning******
- Merging datasets (orders + customers + products)
- Creating calculated columns (total_price)
- Sorting and filtering
- GroupBy and aggregation for analysis
- Date handling (year, month extraction)
- Basic business insights: top customers, revenue by city/category
                     **** INSIGHTS****
- - Top 5 customers by total spending
- Revenue by city and product category
- Average order value and quantity analysis
- Monthly revenue trends
